## HelloWorld  

1. 随便新建一个文件夹，存放代码
2. 新建一个java文件  
   - 文件后缀名为.java
   - Hello.java
   - 【注意点】系统可能没有显示文件后缀名，我们需要手动打开  
3. 编写代码  

```java
public class Hello{
    public static void main(String[] args){
        System.out.print("Hello,World!");
    }
}
```

4.编译 javac java 文件，会生成一个class文件

### 可能会遇到的问题  

1.每个单词的大小不能出现问题，**Java是大小写敏感的**

2.尽量使用英文  

3.文件名和类名必须保证一致，并且首字母大写
